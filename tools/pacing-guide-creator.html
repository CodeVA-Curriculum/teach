Pacing Guide Creator

To use this tool, create one row per unit or lesson you are planning. Then, select the standards you hope to align to for each row. The tool will look at the standards you select and find lessons that match the SOLs you selected to relevant CS-integrated activities!

For now this tool only works with the Virginia Standards of Learning. We are always adding new material and making improvements to the site--please reach out to jonstapleton@codevirginia.org if you have any requests or would like to report a bug. Stay tuned for more lessons and additional support for national standards coming soon!
